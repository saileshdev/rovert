# Rovert

A utility program written in Go which performs image operations.


## Getting started

```
$ git clone https://github.com/saileshdev/rovert
$ cd rovert/
$ export GOPATH=`pwd` 
$ go install rovert
$ bin/rovert
```

Visit "http://localhost:8000" and play around!
